Editor's Note
Hi ELSE! The purpose of this website is to provide a content archive for Pocketdol's group, BAE173!
Of course, this site would not exist without the work of other ELSE and other kpop stans. A lot of the content links were sources from the content archive built by @jjemnyangie on twitter. While I edited the code itself, I used EN-FLIX as a reference for the web design. Beyond this, none of these videos or translations are mine. I really want to give credit wherever I can and I particularly want to highlight our wonderful ELSE translators, so if some of your translations are on this page, please let me know so I can credit you.
I'm not a web designer by any means, but I tried my best to make the site nice and workable. However, I'm always open to new ideas to make the site better - if you have any feedback or ideas as to how to make this better, please feel free to reach out to @echoechoecho173 and I'll try my best to make it happen. Of course, if there are any issues with the site, please don't hesitate to (kindly) let me know as well. I also will try my best to update the site as quickly as possible, but please be patient, as I'm a full-time university student.
FAQ
"How can I help?"
You can help by providing translations! A lot of content isn't translated, so it makes it a bit harder for international ELSE to understand and see their content. I also appreciate it if people let me know if there's any content missing, as I really want to make this archive as complete as possible
"How did you make this?"
I wrote and directly edited the html + css, using the EN-FLIX html as a starting point. If you are interested in a small project learning about website development, I would definitely recommend trying to make a simple website like this, and feel free to reach out and I can try to help as best as I can. ChatGPT is also a pretty good resource for making sense of the code, and figuring out to make your own. Happy coding!
